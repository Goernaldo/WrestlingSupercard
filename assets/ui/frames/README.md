# GCWF UI Frame Asset Contract

Die enthaltenen WebP-Dateien sind einfache rechteckige Platzhalter.

**UI Frame Asset fehlt – später ersetzen.**

Professionelle Frame-Assets können später unter denselben Dateinamen ersetzt werden, ohne HTML, Navigation oder Spiellogik anzupassen:

- `topbar-placeholder.webp` – vollständiger Topbar-Rahmen
- `control-placeholder.webp` – Avatar-, Währungs- und Icon-Control-Rahmen
- `nav-placeholder.webp` – Rahmen eines Navigationseintrags
- `panel-placeholder.webp` – Event-, Mission-, Pack-, Season- und Friends-Panel
- `quick-placeholder.webp` – Schnellzugriffskarte
- `bottom-placeholder.webp` – vollständige untere Statusleiste

Anforderungen an finale Assets: transparenter Hintergrund, WebP oder PNG, keine eingebrannten Texte, gleiche Außenabmessungen beziehungsweise gleiches Seitenverhältnis.
