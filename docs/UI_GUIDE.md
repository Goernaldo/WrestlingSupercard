# UI Guide

Die Farbpalette kombiniert Schwarz und Anthrazit mit Metall-Silber, Rot, Gold sowie sparsamen Cyan- und Violettakzenten. Abstände basieren auf einem Viertel-Rem-Raster. Systemschriften halten die Demo lokal und robust. Flächen sind rechteckig und leicht gerahmt: keine abgeschnittenen Ecken, keine polygonalen Rahmen und keine `clip-path`-Regeln.
